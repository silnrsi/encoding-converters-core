Please change the *.mht files to *.htm

The help files in the <Program Files>\Common Files\SIL\Help folder are stored in the Internet Explorer *.mht (single-file archive) format. But if you want to use Mozilla to display them (by setting the registry key HKEY_LOCAL_MACHINE\SOFTWARE\(Wow6432Node\)SIL\SilEncConverters40\UseMozilla to True), then you must edit the *.mht files (e.g. in Microsoft Word) and save them as *.htm files in the same folder.
